## Installation
```sh
$ yarn install
```

## Start

```sh
$ yarn start
```

After you starting your program, you can remove cowsay that not needed by

```sh
$ yarn remove cowsay
```

## Develop

And start `coding`!
```sh
$ yarn develop
```

## Build
```sh
$ yarn build
```
